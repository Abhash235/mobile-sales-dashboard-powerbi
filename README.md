# 📱 Mobile Sales Dashboard | Power BI

## 📌 Project Overview

The **Mobile Sales Dashboard** is an interactive Business Intelligence dashboard built using **Microsoft Power BI** to analyze mobile phone sales data. The dashboard provides a comprehensive view of sales performance across different brands, models, cities, payment methods, and time periods.

It enables users to monitor key business metrics, identify sales trends, compare monthly performance, and make informed business decisions through interactive visualizations.

---

# 🎯 Project Objectives

- Analyze overall mobile sales performance.
- Track monthly and yearly sales trends.
- Compare sales across different cities and mobile brands.
- Identify the best-selling mobile models.
- Analyze customer purchasing behavior.
- Evaluate customer satisfaction through ratings.
- Understand payment method preferences.
- Build an interactive dashboard for business decision-making.

# 📊 Dashboard Features

## 1. KPI Cards

The dashboard displays important business metrics at the top, allowing users to understand overall performance at a glance.

### KPIs Included

- **Total Sales** – Displays the total revenue generated from mobile sales.
- **Total Quantity Sold** – Shows the total number of mobile phones sold.
- **Total Transactions** – Displays the total number of completed sales transactions.
- **Average Selling Price** – Calculates the average revenue generated per unit sold.

These KPIs automatically update based on the selected filters.

---

## 2. Interactive Filters (Slicers)

The dashboard includes interactive slicers that allow users to filter data dynamically.

### Available Filters

- Year
- Month
- Mobile Brand
- Mobile Model
- Payment Method

Using these slicers updates every chart and KPI simultaneously, enabling focused analysis.
